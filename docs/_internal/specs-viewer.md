# Full Specifications

Interactive viewer for all OpenSpec specifications. Browse requirements and scenarios with filtering by category, spec, and full-text search.

<iframe src="specs.html" style="width:100%; height:900px; border:1px solid var(--md-default-fg-color--lightest); border-radius:8px;"></iframe>

## Regenerating

The specs viewer is generated from OpenSpec markdown files:

```bash
just docs-specs
```

## Source Files

- [OpenSpec INDEX.md](https://github.com/beycom/onetool-mcp/blob/main/openspec/specs/INDEX.md) - Categorized specification index
- [AGENTS.md](https://github.com/beycom/onetool-mcp/blob/main/openspec/AGENTS.md) - Guidelines for AI agents
