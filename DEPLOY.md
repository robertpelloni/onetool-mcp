# Deployment

To deploy OneTool:

1. Copy `.env.example` to `.env` and populate necessary API keys (if applicable). Use `onetool init` and `onetool.yaml` for tool-specific settings.
2. Ensure you have the required system dependencies (e.g. `xclip` or `wl-paste` for clipboard support on Linux).
3. Start the server using `onetool direct start` or your preferred MCP client.
