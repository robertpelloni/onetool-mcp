# Roadmap

## Near-term Tasks
- **Cross-platform Stability:** Ensure robust cross-platform compatibility across all utilities.
- **Error Handling:** Refine error messaging for missing optional dependencies (e.g., Wayland/X11 tools on Linux).
- **Testing:** Expand testing coverage for network and filesystem heavy tools.

## Mid-term Structural Plans
- **Vector Search Expansion:** Expand the `knowledge` pack's vector capabilities and chunking heuristics.
- **Proxy Consolidation:** Further stabilize and consolidate `mcp` proxy integrations to smoothly wrap external tools.
- **Community Packs:** Broaden the ecosystem with community packs.

## Long-term Ambitions
- **Deep Native Integrations:** Deep integrations with more native APIs.
- **Background Processing:** Robust background processing and asynchronous job scheduling.
